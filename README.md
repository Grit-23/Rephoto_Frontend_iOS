## 📱 iOS 개발 담당 파트

### ⚙️ 주요 기능

- **사진 관리**: 업로드한 사진 자동 분류 및 리스트 출력, 태그·설명·앨범 상세 조회
- **검색 기능**: 태그 및 설명 기반 사진 탐색 지원
- **이미지 최적화**: S3에서 가져온 이미지를 NukeUI로 캐싱·로드

---

### 🧑‍💻 나의 역할

- iOS 클라이언트 **전체 설계 및 구현**을 단독 담당
- SwiftUI와 MVVM 아키텍처를 적용해 구조화된 화면과 데이터 흐름 관리 구현
- 백엔드(Spring Boot API)와의 **API 연동**
- Keychain, UserDefaults를 활용해 사용자 정보 및 인증 상태 로컬 관리

---

### 🧩 기술 스택

- **언어/프레임워크**: Swift, SwiftUI
- **아키텍처**: MVVM
- **네트워킹**: Moya, Alamofire
- **이미지 처리**: NukeUI
- **데이터 관리**: Keychain, UserDefaults

---

### 📝 프로젝트 경험 회고

이번 프로젝트는 제가 **iOS 개발 파트를 전담**하여 처음부터 끝까지 직접 구현한 경험이었습니다. SwiftUI와 MVVM을 활용해 구조적인 개발 방식을 익히고, API 연동 및 로컬 데이터 저장을 통해 앱 전반의 흐름을 스스로 완성했습니다.

특히 시니어 사용자를 고려해 접근성과 가독성을 높이는 UX 설계를 직접 적용하며, 단순히 기능 구현을 넘어 **사용자 중심의 서비스 개발**이 무엇인지 체감할 수 있었습니다. 또한 백엔드·AI 파트와 협업하며 API 규약을 맞추고 오류를 해결하는 과정에서 **팀 간 소통과 협업 능력**도 크게 향상되었습니다.

이 프로젝트를 통해 저는 **iOS 앱 전체를 설계·구현**해보았고, 기술적 성장뿐 아니라 **사회적 가치**를 담은 서비스 개발의 의미도 깊이 느낄 수 있었습니다.
